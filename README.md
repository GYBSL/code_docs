# 编程文档

基于 dumi2 

dumi2：[dumi - 为组件研发而生的静态站点框架 (umijs.org)](https://d.umijs.org/)

## 开始

```bash
# 安装依赖
$ npm install

# 开发运行
$ npm start

# build library source code 打包
$ npm run build

# build library source code in watch mode
$ npm run build:watch

# build docs
$ npm run docs:build

# check your project for potential problems
$ npm run doctor
```
